## 🆕 最新：Reading Block

一键保存文章。每存满 5 篇，它就自动在你的日历上排一个 30 分钟的阅读时间块，并把链接都放进日程里，让"稍后阅读"真的发生。📺 [观看演示视频（2 分钟）](https://www.youtube.com/watch?v=Q8g1hod552g)
